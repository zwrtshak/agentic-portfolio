## Selected Projects

### Booking Agency Workflow
End-to-end automation of a DJ booking agency — 9 specialized AI agents (Admin, Booking, Contract, Finance, Logistics, Artist, Promo, Release) coordinated via OpenClaw. Handles the complete booking lifecycle from inquiry to post-event settlement. Fully deployed and operational.

### Healthcare Workflow — Dermatology Practice
Production RAG system for a real dermatological practice. Automated appointment scheduling, AI-driven patient communication, and a Supabase pgvector knowledge base for medical protocol retrieval. Achieved ~30% reduction in administrative workload.

### Hermes Product Development Framework
Multi-agent product management system: a PM orchestrator coordinates Coding, Marketing, QA, and Documentation agents, each equipped with domain-specific skill sets. Obsidian serves as a personal knowledge brain — feeding style, experience, and specs into agent context for consistent, personalized output. Built on Hermes for its superior self-learning and persistent memory capabilities.
